
# HarmonicPattern-Finder
HarmonicPattern-Finder est un outil développé en C# dans le but d'identifier facilement des formes harmoniques (Bullish/Bearish BAT, Bullish/Bearish Butterfly, Reversal...) en utilisant des retracements de Fibonacci dans le cas d'analyse technique de marchés financiers
